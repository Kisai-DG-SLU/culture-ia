# Plan de Développement - Culture IA

## Objectif
Développer un assistant conversationnel RAG pour recommander des événements culturels à partir d'OpenAgenda.

## Étapes
- [x] **Initialisation** : Structure, Git, Environnement Virtuel.
- [x] **Pipeline CI/CD** : GitHub Actions, Linting, Tests Auto.
- [x] **Collecte de Données** : Script ETL vers OpenAgenda, Stockage JSON.
- [x] **Moteur Vectoriel** : Indexation FAISS, Embeddings Mistral.
- [x] **Chaîne RAG** : LangChain, Prompt Engineering, LLM Mistral.
- [x] **API Backend** : FastAPI, Endpoints /ask et /rebuild.
- [x] **Tests & Qualité** : Pytest, Coverage > 70%, Évaluation Ragas.
- [x] **Déploiement** : Dockerisation optimisée (Multi-stage build).
- [x] **Documentation** : README complet, Rapport technique, Slides.
